Introduction

This site allows you to compare result data of a filament analysis series conducted by Thomas Sanladerer on his YouTube channel.
You may watch his introduction to the series using this link.

It is developed and maintained by Sebastian Walker.

Data

All analysis data has been extracted from Thomas' videos and formatted to JSON.

You may access the JSON file this site reads from using this link. Thomas has also put together a Google Docs sheet with the complete data.

If you encounter any incorrect data, please shoot me an email at sebastian@sebastianwalker.org.

Techie Stuff

This site is powered by the Vue.js library. It uses Bulma as a UI framework.
Favourites are stored in localStorage.

Pretty simple, huh?
